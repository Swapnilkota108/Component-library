Button

Buttons are used to perform some action.

Simple Buttons

Simple buttons can be created using btn class followed by btn-specificity.

Outlined Button

Outlined buttons can be created using btn class followed by btn-outline-specificity.

Link Button

Link buttons contains link to some other page. Link buttons can be created using btn followed by btn-link.

Button with Icon

Button with a icon can be created using btn class followed by btn-icon class.

Floating action button

Floating buttons can be created using btn-float class.
