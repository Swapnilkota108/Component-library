Alert

Alert are used to prompt feedback messages to the user.

Simple Alert

Simple Alerts comes in 4 styles, To apply proper styles use the contextual classes for the alert.

Primary! This is a Primary Alert.
Error! This is an Error Alert.
Success! This is a Success Alert.
Warning! This is a Warning.
